# CODING STANDARD

## Plataforma oficial

- C23
- MSYS2 UCRT64 GCC
- CMake
- Ninja

## Reglas obligatorias

- Cero errores y cero warnings.
- Prohibido silenciar diagnósticos para ocultar defectos.
- Sin comportamiento indefinido, fugas de memoria ni hacks temporales.
- APIs con contratos explícitos y resultados verificables.
- Corrección de causa raíz acompañada por prueba de regresión.
- Código portable dentro de la plataforma oficial y sin dependencias innecesarias.

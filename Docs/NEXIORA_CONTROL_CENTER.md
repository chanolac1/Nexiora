# Nexiora Control Center

Ejecutable: `Build/windows-msvc-release/bin/nexiora_control_center.exe`

La interfaz ofrece navegación lateral, formularios contextuales, progreso observable y salida en tiempo real.

## Flujos

- **Aprender archivo:** selecciona la capacidad, escribe la ruta y ejecuta.
- **YouTube:** escribe la URL, el idioma y ejecuta.
- **Herramientas:** consulta el estado de `yt-dlp` administrado.
- **Pruebas:** ejecuta CTest completo.
- **Paquetes:** consulta historial por Package ID.
- **Documentación:** ejecuta la validación documental.

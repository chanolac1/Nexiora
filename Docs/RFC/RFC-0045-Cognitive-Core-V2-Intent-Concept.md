# RFC-0045 — Cognitive Core v2: Intent and Concept Grounding

## Decisión

Las respuestas cognitivas deben separar intención, concepto principal y evidencia usada.

## Regla

Si hay coincidencias directas del concepto principal, se usa solo esa evidencia. Las relaciones laterales se usan únicamente en preguntas comparativas o explícitamente relacionales.

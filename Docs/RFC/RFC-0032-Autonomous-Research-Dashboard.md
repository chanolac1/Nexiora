# RFC-0032: Autonomous Research Dashboard

## Status
Accepted

## Summary
EPIC-0006 introduces a static HTML dashboard for autonomous research sessions.
The dashboard is intentionally local-first and generated from session artifacts.
It does not promote runtime changes and it does not require a server.

## Principle
The dashboard is a visualization layer over evidence. It must not become a decision authority.
Nexiora may recommend; only a human may promote.

## Artifacts
The dashboard links to summary, report, metrics, knowledge and graph exports generated by `nexiora research run`.

# RFC-0034 — Research Progress Framework

Toda operación de Nexiora que pueda tardar más de dos segundos debe reportar progreso observable.

## Estados

- PENDIENTE
- EN PROCESO
- OK
- ADVERTENCIA
- ERROR
- CANCELADO

## Principio

Nexiora no debe permanecer en silencio durante tareas largas. Debe comunicar paso actual, progreso, última actividad y finalización.

# RFC-0033 — Conversación con el Laboratorio

EPIC-0007 introduce una consola conversacional en español para Nexiora.

La consola no es un chatbot genérico. Es una interfaz disciplinada hacia el laboratorio de investigación. Cada respuesta debe ser conservadora, explícita y basada en el estado que Nexiora conoce.

Principios:

- No inventar conclusiones.
- Mantener aprobación humana para toda promoción.
- Responder con confianza estimada cuando aplique.
- Preferir decir "todavía no sé" antes que simular certeza.
- Permitir que Jorge dialogue con el laboratorio mediante comandos naturales en español.

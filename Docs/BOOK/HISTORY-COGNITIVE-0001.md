# HISTORY — COGNITIVE-0001

Se introduce el primer Cognitive Core funcional:

- Ingesta de archivos locales.
- Extracción de conceptos.
- Fragmentación de texto.
- Preguntas sobre memoria local.
- Registro metadata-only para media sin extractor disponible.

Este hito permite comenzar ejercicios funcionales con archivos reales antes de añadir OCR, video, PDF o web avanzada.

# DECISIONS

## ADR-0002 — Documentación como contrato transaccional de entrega

**Estado:** Aceptada  
**Fecha:** 2026-07-10

Nexiora mantiene PROJECT_STATE.md, CHANGELOG.md y DECISIONS.md como una sola transacción. Una entrega se rechaza cuando falta documentación obligatoria o una referencia de código declarada no existe.

# Qué logramos — Cognitive Core v2 Sprint-01

Nexiora puede representar un objetivo grande como un grafo jerárquico auditable, declarar dependencias, identificar la siguiente tarea ejecutable y evitar completar trabajo fuera de orden. Esta base permite evolucionar hacia planificación de proyectos complejos, incluyendo videojuegos de acción RPG.

## Evidencia
- Formato persistente `nxgoalgraph/1`.
- CLI `nexiora_goal_graph.exe`.
- Prueba `NxGoalGraphTests.exe`.
- Detección de ciclos y transiciones inválidas.
- Orden determinista por prioridad e ID.

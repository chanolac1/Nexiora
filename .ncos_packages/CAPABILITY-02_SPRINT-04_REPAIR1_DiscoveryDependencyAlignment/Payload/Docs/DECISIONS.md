# DECISIONS

## ADR-0024 — Los vacíos de conocimiento son artefactos persistentes

**Estado:** Aceptada.

Nexiora no debe improvisar cuando la evidencia es insuficiente. Debe registrar un artefacto `nxgap/1` con pregunta, razón, consultas, fuentes autorizadas y criterio de cierre. La investigación autónoma futura consumirá este contrato.

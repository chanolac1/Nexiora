# UI-001 — Nexiora Control Center

## Qué logramos

Se incorpora una interfaz nativa de Windows para operar Nexiora desde una sola ventana.

Capacidades iniciales:

- aprendizaje desde archivos;
- aprendizaje desde YouTube;
- consulta del Tool Manager;
- ejecución de pruebas;
- historial de paquetes;
- validación documental;
- progreso visible y consola integrada;
- ejecución asíncrona para evitar congelar la interfaz.

La interfaz no reemplaza las herramientas CLI. Las orquesta mediante procesos separados y conserva su salida completa como evidencia.

# NCOS-025 — Qué logramos

Nexiora genera evidencia de certificación machine-readable y legible después de cada Apply.

## Compuertas
- Build y cero warnings.
- Suite completa repetida para hermeticidad.
- Validación de artefactos obligatorios.
- Documentación sincronizada.
- Reporte aun cuando existe rollback.
- Recomendación explícita `APPROVE_GIT` o `DO_NOT_COMMIT`.

## Regla de publicación
El usuario comparte `certification-report.nxcert`; Arquitectura y QA revisan el archivo antes de proporcionar instrucciones Git.
